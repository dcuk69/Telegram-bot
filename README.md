## Getting Started

### 1. Create a bot with BotFather

1. Open Telegram and search for **@BotFather**. [web:3]  
2. Send `/newbot` and follow the prompts to set a name and username. [web:3]  
3. Copy the HTTP API token that BotFather returns. [web:3]
